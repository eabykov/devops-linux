# Задания по Linux для Айдера

### Первое задание

1. Перейди в `/tmp` директорию
2. Вернись обратно в директорию в которой был первоначально
3. Перейди в домашнюю директорию
4. Перейди в вышестоящую директорию
5. Перейди в корень (что такое корень?)
6. Выполни команду `cd ~`
7. Какой полный путь к директории где ты сейчас находишься?

<details>
  <summary>Решение первого задания</summary>

  1. `cd /tmp`
  2. `cd -`
  3. `cd`, `cd ~` или `cd $HOME`
  4. `cd ..`
  5. `cd /`
  6. `cd ~` - перейдем в домашнюю директорию
  7. `pwd`

</details>

### Второе задание

1. Создай файл с именем `chlen`
2. Создай директорию `boobs`
3. Перемести файл `chlen` в директорию `boobs`
4. Создай файл `sperm` в директории `boobs`
5. Создай в директории `boobs` подпапки `conchat/conchat/conchat` одной командой
6. Вернись в самое начало и удали директорию `boobs`

### Третье задание

1. Создай пустой файл `file.txt` в директории `/tmp`
2. Скопируй `file.txt` файл в свою омашнюю директорию
3. Создай копию файла `file.txt` которая будет называться `txt.file`
4. Создай директорию `govno` и перемести в нее файлы `file.txt` и `txt.file`
5. Создай копию директории `govno` которая будет называться `ne_govno`
6. Переименуй `ne_govno` в `govno_govno`
7. Удали обе директории которые создал и все файлы одной командой

### Четвертое задание

В директори есть файл в котором написан список IP адресов, вычисли какой из них чаще всего повторяется

### Пятое задание

Вывести только время когда Айдер запускал команду `sudo apt update` в файле /var/log/auth.log
